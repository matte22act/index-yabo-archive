# index-yabo-archive

本仓库用于归档和发布多个独立的 HTML 页面。  
这些页面来自不同的历史版本或临时用途，现集中存放于此，便于统一管理和访问。

## 项目简介

index-yabo-archive 是一个静态页面归档仓库。  
仓库内包含若干 HTML 文件，每个文件均为独立的网页，不依赖于任何后端服务。  
所有页面均以静态形式提供，可直接通过浏览器打开或部署至静态托管平台。

## 目录说明

```
/
├── index.html          # 主入口页面（如有）
├── page1.html          # 示例归档页面
├── page2.html          # 示例归档页面
└── assets/             # 页面引用的资源文件（如 CSS、图片等）
    └── ...
```

- 根目录下的 `.html` 文件即为归档的独立页面。
- `assets/` 目录存放页面所需的公共资源，如样式表、脚本或图片。
- 各页面之间无耦合关系，可单独使用。

## 页面归档说明

- 每个归档页面均保留其原始结构和功能，未做额外修改。
- 部分页面可能包含外部资源引用（如 CDN 或外部图片），访问时需联网。
- 归档内容仅供查阅与学习，不保证与原始环境完全一致。

## 维护说明

- 本仓库由维护者不定期更新，新增的归档页面会添加至根目录。
- 若发现页面无法正常显示或存在链接失效，欢迎提交 Issue 反馈。
- 不接受针对具体页面内容的修改请求，仅处理归档结构或访问性问题。
